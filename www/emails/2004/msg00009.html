Wow! How cool is that?

We started playing w/ Test::Simple last year. Pretty slick.

So, I'm in a directory with 6 .t files wondering if there's a neato way to
run all those test files in one fell swoop. Of course, there was. 10
minutes later I'm using Test::Harness to run all my test files.

(I ran Test::Harness across our entire OO Perl structure and 87.43% of our
517 subtests worked. Yay!)

Wow.

j

perl -e 'use Test::Harness; runtests(@ARGV)' `find ./ -name "*.t"`
