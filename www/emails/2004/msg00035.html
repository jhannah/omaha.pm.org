Hi Daniel:

First observation: Compressed binary streams aren't easily editable.
It is possible, if you can edit frames directly, to cut in pieces but not
much else.  For fading in and out, though, you'll have to re-encode.
These aren't tasks that belong to Perl, per se.

Perl module to handle resampling:
   http://search.cpan.org/~lds/Apache-MP3-3.05/MP3/Resample.pm

I think the first question you should ask yourself is what library you
want to be using for re-encoding and/or editing of the MP3 format.
Here's a good starting place.
  http://lame.sourceforge.net/links.html

Sox can give you a fadeout taper after you've got it uncompressed.

For clipping/cutting without re-encoding, see:
  http://mp3asm.sourceforge.net/

MPEG Frame module is the closest pure-perl code that is relevent to
the problem you have proposed to solve:
  http://search.cpan.org/~nuffin/MPEG-Audio-Frame-0.08/Frame.pm

Please let me know how it turns out.
