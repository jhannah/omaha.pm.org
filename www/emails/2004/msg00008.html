At 10:41 PM 1/5/2004 -0600, Jay Hannah  - jay@xxxxxxxx wrote:
>Howdy, "tech@xxxxxx"...

Well, that's not me, but.... 

>Is there any way we could get Majordomo configured w/ a Subject line 
>tag? Like "[Omaha.pm] " or something?

Yup. 

Send a message to majordomo@xxxxxx with just the following line in the body
of the message:

config omaha-list <your password here> 

making the appropriate substitution, of course. 

You will get a rather long message back from majordomo. Redirect it back to
majordomo. Do not "reply." 

At the top of the message change the command 
config omaha-list <your password here> 
to 
newconfig omaha-list <your password here> 

At the bottom of the message, remove the update date and type 
EOF
on a line by itself immediately following the last line in the message. 

Then search for the phrase "subject_prefix      =" in the message. When you
find it, type in the tag you want for the list. 

I suggest as strongly as possible you keep it short, prefereably 3 letters,
and enclose it in [ ] 

Then go look for the phrase "message_headers     <<  END" which is probably
immediately followed by "END" on a line by itself (aka a here document)

Between the two lines, on a line by itself, type: 
X-Loop: $LIST email list

Send the message. 

You should get a message back saying the new config for omaha-list was
accepted. 

Then those of us who care to can filter on the X-Loop header instead of the
subject line tag {spit}.

-Sidney
