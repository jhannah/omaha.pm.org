> 2) a cronjob that checks every 1 or 2 hrs,  if this script is running, if not, starts it up.

Hi Babu --

1) What flavor of *nix is it? If you have a file /etc/rc.d/rc.local, you
may want to add "/usr/bin/perl /home/babu/filename.pl" (correct the
paths) to that file to get it to start on boot.

2) I spent 10 minutes writing a crude little Perl program that looks for
a process and starts it if it doesn't find it. Should work on most
*nix's, I think. You may need to change where Perl lives if it's not
/usr/bin/perl on your system.

---
#!/usr/bin/perl

my $program = "/usr/bin/perl /home2/jhannah/loop.pl";
my $foundit = 0;

open (IN, "ps axw |");
while (<IN>) {
   if (/$program/) {
      $foundit = 1;
      last;
   }
}
close IN;

unless ($foundit) {
   system("$program &");
}
---

I hope that helps,
