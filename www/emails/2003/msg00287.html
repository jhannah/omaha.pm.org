Thanks Jay.

I was able to get the output how I wanted by using <TMPL_LOOP> and <TMPL_IF>.  i will give the solution here so that it might help somebody.

TMPL_IF only works with parameters, so I thought the parameter name should be given within the param() braces. there is a workaround. I created a hash with key as the parameter name that I wanted and then pushed the hash into an array and sent the array to the template. In the template file I used TMPL_LOOP.  I was able to catch the parameter with <TMPL_IF name=parametername> but because the array passed all values to the <TMPL_LOOP>,  i had to dummy other parameters that I dont want with empty <tmpl_if> tags.

Babu
