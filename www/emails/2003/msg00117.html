* thehaas@xxxxxxxxxx (thehaas@xxxxxxxxxx) wrote:
> Is there anyway I can find out if a scalar value I have is an element of
> an array without looping through the whole array looking for it?

depending on what you're doing you may want to use a hash.  

    my @array = qw(apple bananna orange pear);
    foreach my $fruit (@array) {
	# do something
    }

and you want to test if pear is in the list, you'd have to go through
the whole list to find out.  this isn't really a problem unless you have
a HUGE array, then you should maybe use a hash

    my %hash = (
	apple   => 0,
	bananna => 1,
	orange  => 2,
	pear    => 3,
    );

so you would test like this

    if (exists $hash{pear}) {
        # do something
    }

and you could still do array like this like this

    foreach my $element (sort $hash{$a} <=> $hash{$b}) keys %hash) {
        # do something
    }

It all depends how big your array is really.  One other alternative
would be to make a hash from your array so you have copy (depending on
memory considerations etc)

    my @array = qw(apple bananna orange pear);
    my $count = 0;
    my %hash = map { $_ => ++$count } @array;

Hope this helps and anyone correct me if something doesn't seem right.
