> From: "ktegels" <ktegels@xxxxxxx>
> I had an RegExHelper instance today too. Somebody wanted a RegEx for a
> 1/10th-limited decimal number.
>
> \d+\.\d?

I think what you may want is:

   /^\d+\.\d$/

Breaking that down:

/^     the string begins with
\d+    one or more digits
\.     followed by a literal period
\d     followed by exactly one digit
$/     and that's the end of the string

(That's the regex in Perl land anyway...)
