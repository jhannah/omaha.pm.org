It would be beneficial to learn some win32 things in conjunction with Perl to
really garner the benefits of Perl in Bill's World. If you have time and are 
serious about using Perl in Win32 these links might help:

Win32 API

http://msdn.microsoft.com/

ADSI

You can download ADSI help files here:

http://www.microsoft.com/adsi/ 

Delve into here:

http://msdn.microsoft.com/library/

Click into on the left nav section:

Network and Directory Services

then

Active Directory, ADSI and Directory Services

Looks like one would need to do some rooting at this point. But there is
lots of good documentation on the exciting world of adsi.
