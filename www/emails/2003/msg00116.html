FWIW, my current habit is:

if (isInList($thing,@list)) {}

sub isInList {
    my $lookFor = shift;
    foreach (@_) { return 1 if ($_ eq $lookFor) }
    return 0;
}

As Jay already pointed out, this isn't a good idea for large lists.
