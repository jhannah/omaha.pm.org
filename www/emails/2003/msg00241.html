* omaha.pm.knitter@xxxxxxxxxxxx (omaha.pm.knitter@xxxxxxxxxxxx) wrote:
> It's possible, if you don't have root, to install perl and assorted
> modules. It isn't fun. You cannot use cpan. You cannot use the general
> instructions (download the file, untar it, and issue perl makefile.pl,
> make, make test, make install.) if you can't use cpan. And you cannot use
> the instructions given for virtual installation because they were written a
> couple of years ago for a different system and don't work. 

When you install modules you need to paths defined

    one where you want your local stuff installed - PREFIX (just like C)

	PREFIX=/home/jbisbee/local
	
    and one where you want your perl stuff installed - LIB

	PREFIX=/home/jbisbee/local/lib/perl5

Then set the PERL5LIB environment variable in your base config

    PERL5LIB=/home/jbisbee/local/lib/perl5

and in your scripts

    use lib '/home/jbisbee/local/lib/perl5';

Now installing stuff from CPAN you'll need to do

    perl Makefile.PL LIB=/home/jbisbee/local/lib/perl5 PREFIX=/home/jbisbee/local
    make
    make test
    make install
