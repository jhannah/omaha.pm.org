We're all too busy reading the Cookbook, I guess :)

Given that there isn't much going on with this list, I'll try this for a
while. Once a week, I'll pose a programming puzzle for the group to hack on.
And what do you get for your hard work?  Well aside from the satisfaction of
solving a (hopefully) worthy puzzle and sharing that with many other Perl
Mongers, I'll post the best (in my slanted, twisted and biased opinion)
solution on my web site for a while giving you the credit you've earned.

So without additional adieu... :

You've just inherited the administration of an intranet web that has
consists of bunch of html files in many deeply directories. A number of
these files reference a Java application called Puzzle1. Developers, being
prone to "enhance things," have decided to add new functionality to the
applet as driven by a parameter called p3. p3's default value should be
(guess?) "MyValue3."

<applet code="puzzle1.class" width=250 height=50
codebase="jars/puzzle1.java">
<param name="p1" value="myValue1">
<param name="p2" value="myValue2">
<p><font color="#E70000">Get a clue. Get Java.</font></p>
</applet>

The developers, needing to work on other "enhancements," have asked you to
"just toss up a little script" that reads through all the files in
/webs/intranet/puzzle1 (\inetpub\wwwroot\puzzle1 for the Windows-centric)
and the sub-directories thereof. You need to rewrite any occurrence of this
<applet></applet> set to include the new parameter.

Oh, I nearly forgot. You need to write this application so you can run it
either the production server, A FreeBSD 2.2.7 box and on the staging server,
a Windows NT server 4. Assume all you have is Perl on either machine, no
Grep, no Sed, no Awk.

The "best" solution will be the fewest total non-white space characters that
works correctly. To encourage you to comment your code so we can all
understand it, the number of non-whitespace characters in comments (defined
as roughly as /^\#.*\\n$/) won't be included in the scoring unless I decide
to apply them as some sort of tie breaker.

For extra credit, keep a log of the files you've changed as a text file.

E-mail your solutions to kent@xxxxxxxxxx or to the list by 11:59:59 Local
System Time on 1998.12.19 I'll probably select and post "the best" within 96
hours.

Good luck,
Kent
