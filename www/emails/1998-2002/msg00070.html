In macPerl these executable  are called Runtime Versions and you can do the
following:

	#!perl
		require "XXX.pl";
		exit( 0 );

And then put your source code into XXX.pl.  This allows you
to have a Runtime Version which is quite a bit smaller.

I don't know if it will work that way on any other platform.
