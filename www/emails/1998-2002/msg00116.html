On Mon, 13 Dec 1999, Ryan wrote:

> Hello. I'm looking at upgrading a RedHat 4.2 box to 6.1.
> Basically we're using a fresh install of 6.1 and bringing
> all our custom stuff over, (web pages, radius, user directories,
> etc). My question is, is there an easy way to tell what
> perl modules we've installed on the old box so I know what
> to install on the new one? I've looked in the include
> directories, and it gives some clue, but a lot of stuff
> in there comes with perl. Any ideas much appriciated. 

My suggestion is to take a look at the file called perllocal.pod
perldoc perllocal
(for me, it was in /usr/libdata/perl5/i386-openbsd/5.00503) 

I'm not sure if it is a complete list of what is installed as it appears to be
a list of documentation included with the installed modules, so if a module
didn't provide documentation it might not be listed.

I'd also check out your site_perl directory, probably in 
/usr/lib/perl5/site_perl on your redhat box.

Hope that helps.

-Rob
