 Drazak wrote:

 > I have an referenced array that I can loop through and print out the
 > values with this:
 >
 > print @$_;
 >
 > However, I just want the first three chars of the array.  Wrapping it
 > with a substr just produces the value "1", as does trying to assign the
 > value to a new scalar.  I've been reading through perlref with no luck,
 > and I left both of my Oreilly books at home.

An easy way to do this is with my favorite perl function 'map'

Assuming $_ holds a reference to an array,
(ie:
my @Array = ('abcdefg', 'hijklmn', 'opqrstu');
my $rArray = \@Array;
$_ = $rArray;
)

map { print substr($_, 0, 3) . "\n"; } @$_;

will print out the first three characters of each element of the array.

similarly if you wanted to assign the first three characters into another
array you could say:

my @NewArray = map { $_ = substr($_, 0, 3); } @$_;

Once you understand map you can really create some powerful (and quite
possibly obfuscated) code.
