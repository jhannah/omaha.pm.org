> Hi I've put together this little script to search a file and it works
> fine but I can't get it to print out a statement such as, "Sorry match
> not successful." if in fact the search returns nothing.  At this point
> the program just exits if the search is not successful.
> 
> Here is the relavant piece of code, I will include the full program
> below:


This is the solution that came to mind...  Create a new variable alled
$found, and set it to zero before you go into the while loop.  When you
find a match, set the value to '1'.  When the while loop is done, $found
will have a value of '1' if there was a match, or it will still have a '0'
if there was no match.  So, you check to see if $found is equal to zero,
and if it is, print your message.  Like this:
 

$found = 0;

 #####
 while (<BOTTLEFILE>) 
 {
    if (/\Q$SearchString/i) #If the search word is found.
    {
        print "\n$_\n";   #..print the output.
 	$found = 1;
    }
 }
 #####

if (!$found)
{
   print "Sorry, the string \'$SearchString\' was not found.\n";
}



That should work.  Of course, there are as many ways to do it as there are
perl geeks.  :)  Good luck,
