I always use the FileHandle module for file operations.  It looks more like
'C' to me (what I was used to).

Here is an example.
~~~~~~~~~~ start code ~~~~~~~~~~~~~
use FileHandle;

my($Fh) = new FileHandle("./temp.txt", "r");
if (defined($Fh)) {
  my($InLine);
  while ($InLine = $Fh->getline()) {
    print($InLine);
  }
  undef($Fh);             ## close the file
}
~~~~~~~~~~ end code ~~~~~~~~~~~~~
