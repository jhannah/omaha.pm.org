An Omaha Java Users' Group is forming.  Do you think this is a good
idea?  What kind of things would you like to see the group talk about?

Markus Kobold <mrperkins@xxxxxxxx> called an organizational meeting on
12/11/1999.  The group is still in the formative stages, but we will
start having meetings.   

The next meeting will be:
    What: Omaha Java Users' Group meeting
	  Kent Smotherman will speak on 
	       - Basic use of Object Serialization
	       - Core Reflection & Introspection (relates to the above)
	  Organizational stuff (and perhaps lunch!) after Kent's talk      
    Who:  Everyone is welcome
    When: Saturday January 8th, 2000 starting at 10am.
    Where: Creighton University 2500 California Plaza - Omaha, NE 68178
    Why:  Have *FUN* with Java and other Java enthusiasts
    How:  The powers-that-be are allowing us to use a great multimedia
          classroom at Creighton University.  Directions will be sent
          soon.
    Webpage: http://OJava.pinko.net

Future talks in the works include Swing, Servlets, Network
programming, and UML.  What are future talks you would like to see?  What
companies might be interested in talking?  Which compaines would you like
to hear present?

To subscribe to the Omaha Java Users' Group send an email with
"subscribe" in the subject line to:
    OJava-request@xxxxxxxxxxxxxx

Please subscribe to the mailing list.   Future emails will only be 
sent to the list.

To subscribe to the Omaha Java Users' Group announcement-only list
send an email with "subscribe" in the subject line to:
    OJava-Announce-request@xxxxxxxxxxxxxx

To unsubscribe, put "unsubscribe" in the subject line.

Please join us!  Please encourage others to join us!
