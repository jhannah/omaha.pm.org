Thanks George & Bob!!!   In the stock CGI.pm Animal crackers example
it says right on the page that it's not compatible with IE.  But
adding two lines from Bob's example made it work with IE4!
# Add new animals to old, and put them in a cookie
$the_cookie = cookie(-name=>'animals',
                     -value=>\%zoo,
                     -path=>'/~payne/',          # Added
                     -domain=>'payne.pinko.net', # Added
                     -expires=>'+1h');

Thank you **VERY** much! -Matt

On Fri, 5 May 2000, Robert Fulkerson wrote:

> Matt,
> 
>    http://squishy.ist.unomaha.edu/~robert/class2.cgi
> 
>    The first time you load it, it won't print anything, because the cookie 
> is being set.  When you reload it, it should display the cookie and it's 
> value.
> 
> Here's the very simple code:
> 
> #!/usr/bin/perl
> # http://squishy.ist.unomaha.edu/~robert/class2.cgi
> 
> use strict;
> use CGI qw(:all);
> 
> my $query = new CGI;
> 
> my $cookie = $query->cookie(-name=>'bobID',
>                             -value=>'xyzzy',
>                             -expires=>'+1h',
>                             -path=>'/~robert/',
>                             -domain=>'squishy.ist.unomaha.edu') ;
> 
> print $query->header(-cookie=>$cookie);
> 
> foreach ( $query->cookie() ) {
>       print "Well, cookie '$_' appears to be " . $query->cookie($_) . p;
> }
> 
> ----------------------------------------------------------------
> 
> Or, for doing multiple cookies:
> 
>    http://squishy.ist.unomaha.edu/~robert/class3.cgi
> 
> Here's the code for setting multiple cookies:
> 
> #!/usr/bin/perl
> # http://squishy.unomaha.edu/~rfulk/class3.cgi
> 
> use strict;
> use CGI qw(:all);
> 
> my $query = new CGI;
> my $cookie = $query->cookie(-name=>'bobID', -value=>'xyzzy',
>                             -expires=>'+1h', -path=>'/~robert/',
>                             -domain=>'squishy.ist.unomaha.edu') ;
> my $cook2 = $query->cookie(-name=>'Huh', -value=>'yupyup',
>                            -path=>'/', -domain=>'.unomaha.edu');
> 
> my $cook3 = $query->cookie(-name=>'HelpMePlease', -value=>'YoYo',
>                            -path=>'/', -domain=>'.unomaha.edu');
> print $query->header(-cookie=>[$cookie,$cook2,$cook3]);
> 
> foreach ($query->cookie() ) {
>       print "Well, cookie '$_' appears to be " . $query->cookie($_) . p;
> }
> 
> ----------------------------------------------------------------
>
