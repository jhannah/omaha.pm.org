I guess the cheater way (a.k.a the way I'd do it :) would be to use the
'setleds' command that comes with linux, and run it from your PERL script.
I'm a fan of using backticks to run system commands, So I'd do something
like...

----
#!/usr/local/bin/perl

`setleds +num`;

#Rest of your script....
------

You'll have to check the man page to see if that will work for what you
want to do...  Let me know I'm way off base.  Have a good one,
