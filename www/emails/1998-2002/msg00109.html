This easiest way I can think of is to use a flag:

#####

my $match_found = 0;

while (<BOTTLEFILE>) {
	if (/\Q$SearchString/i) {
		print "\n$_\n";
		$match_found++;
	)
}

print "Sorry, match not successful.\n"
	unless ($match_found);

#####
