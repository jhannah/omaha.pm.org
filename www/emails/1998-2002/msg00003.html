Just enlisted.

I'm a proud MacPerler. Any other MacPerlers out there?

Most of my Perl code experience has been in processing html forms,
accessing databases, etc. for Unix servers.

The reason I learned Perl originally was because of its cross platform
compatibility and its suitability for use on the internet but I keep
slipping further and further off track.

Lately I've been exploring MacPerl modules that allow access to many of the
Macintosh System's toolbox routines. I've been designing MacPerl programs
for my personal use, mainly to enhance my productivity.

Two or three weeks ago I discovered that the module that allowed access to
the Macintosh speech routines provided a way to control the pitch of the
speakers.

Soon I had MacPerl singing a scale (do, re, mi, etc.). A few days later I
had MacPerl singing Camptown Races. Somebody on the MacPerl list modified
the code so MacPerl could sing sing Daisy Bell, since that's what Hal sang
on "2001 Space Odyssey".

My latest effort was to produce a MacPerl harmonica simulator called
"Harmonica Helper". You play the on-screen-harmonica using the number keys
on your computer keyboard. The computer teaches songs one phrase at a time.
First the blow note holes turn blue and the draw note holes turn yellow as
a phrase of the song sounds. You must repeat the pattern on your number
keys using the shift key for draw notes.

I wrote the first version of "Harmonica Helper" for the Commodore 64 back
in 1986 or so. It ran at the Omaha Children's Museum for a couple of
years(?).

I'd like to do a version for Windows and was hoping there'd be a way to do
it with Perl.

Has anyone out there been accessing Windows 95 specific routines,
especially music capabilities?

I run SoftWindows 95 on my rather low end PowerPC mac compatible and have
tested some of my Perl code with it to make sure it was compatible. I'm
sure I must be using an outdated version... Perl for Win32 - 5.001m Build
110 from hip communications inc. Nevertheless, it has been satisfactory
for my purposes.
