Drazak wrote:

> I have an referenced array that I can loop through and print out the

>

> value to a new scalar.  I've been reading through perlref with no luck,

The first three characters of the array?  Your NOT thinking in 'C', are
you ?!!!! :)

Remember in Perl you can deal with a string as a single scaler value ...
you don't have to put each character of the string into an array in
order to deal with it. With your input in a string, you could get at the
first three characters like this:

#!/usr/bin/perl

$string = 'Pattern';
$str    = substr $string, 0, 3;
print $str . "\n";


Another way, this time using a regular expression (again, assuming your
input is in a scaler,  not an array):

#!/usr/bin/perl

$string = 'Pattern';
($first,$second,$third) = ($string =~ /(.)(.)(.).*/);
print $first . $second . $third . "\n";


But, to try to answer your question, using an array as input:

#!/usr/bin/perl

@arr = qw(P a t t e r n);
# create our reference to the array, as you requested:
$r_arr = \@arr;                       
# to access the array itself, we'll have to de-reference it with the
# "@{ $r_arr }"; then we'll take a slice of the de-referenced array
# using the "[0..2]" :
($first,$second,$third) = @{ $r_arr }[0..2];
print $first . $second . $third ."\n";
